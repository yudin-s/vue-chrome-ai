# Changelog

## 0.1.0

- Initial Vue 3 composables and plugin for Chrome's browser-side `LanguageModel` API.
- Port framework-neutral runtime helpers from `@yudin-s/react-chrome-ai`.
- Add Prompt API availability, params, session lifecycle, download progress, prompt, streaming, append, clone, context, structured output, and reflection helpers.
- Add task API composables for Summarizer, Translator, Language Detector, Writer, Rewriter, and Proofreader.
- Add Vue-native reactive option support for composable/session/task settings.
- Add TypeScript declarations, ESM/CJS builds, docs, recipes, examples, tests, CI/publish workflows, package provenance config, and npm metadata.
